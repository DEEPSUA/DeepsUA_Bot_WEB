DEEPSUA TELEGRAM BOT - Documentación¶

Documentación del bot de Telegram de DEEPSUA. Podemos encontrar el bot en Telegram buscando @deeps_ua_bot o en el siguiente enlace: DeepsUA_Bot.

Comandos disponibles¶

Los comandos que se encuentran disponibles en el bot son los siguientes:

Comando Estado Descripción
/addadmin OK Este comando añade un administrador al bot, que podrá acceder a las funcionalidades que requieran elevación
/borramedecomision OK Este comando te permite borrarte de una comisión
/cancel OK Este comando cancela la ejecución de un comando
/comisiones OK Este comando devuelve una lista con las comisiones que existen en la Delegación
/coordinadores OK Este comando llama a todos los administradores de un grupo
/deepsrrss OK Este comando devuelve las redes sociales de la Delegación
/fingerrules OK Este comando devuelve el PDF de las fingerrules
/inscribeme OK Este comando te inscribe en una comisión
/llamarcomision OK Este comando sirve para llamar a los miembros de una comisión en un grupo
/nuevapublicacion DEV Este comando permite subir publicaciones a redes sociales
/vercomisiones OK Este comando sirve para ver en qué comisiones está el usuario citado (sólo admins) o para ver en qué comisiones está el usuario que lanza el comando
/verusuarioscomision OK Este comando permite ver los usuarios que se encuentran en una comisión

Instalación de librerías¶

Para el correcto funcionamiento del bot de telegram, será necesario que instalemos las siguientes librerías:

pip install pytz
pip install python-dotenv
pip install python-telegram-bot
pip install tweepy

También podemos instalar las librerías con el siguiente comando:

pip install -r requirements.txt

Del archivo install.sh¶

Este archivo sirve para instalar el bot en nuestro sistema linux. Esto no se debe de ejecutar a no ser que querramos instalar el bot en un nuevo servidor. Tener varias instancias del bot producirá fallos, por lo que si queremos probar algún comando, se tendrá que hacer en local y bajo consola, sin lanzar el bot. Este archivo crea un fichero de servicio de SYSTEMCTL, lo habilita y lo lanza.
