DEEPSUA Bot - Handlers¶

Los handlers se encargan de gestionar los mensajes que recibe el bot. Cada comando tiene su propio handler. Los handlers se encuentran en el archivo deepsua_handlers.py.

Cabe destacar que tenemos distintos tipos de handlers:

  • CommandHandler: Este tipo de handler se encarga de gestionar los comandos que se le envían al bot. Por ejemplo, el comando /start o /help.
  • MessageHandler: Este tipo de handler se encarga de gestionar los mensajes que se le envían al bot. Por ejemplo, los mensajes de texto o los mensajes de tipo sticker.
  • CallbackQueryHandler: Este tipo de handler se encarga de gestionar las respuestas a los botones de los mensajes de tipo InlineKeyboardMarkup.

Start¶

El comando /start es el comando que se ejecuta cuando un usuario inicia una conversación con el bot. Este comando se encarga de mostrar un mensaje de bienvenida al usuario.

Cabe destacar que el handler de este comando es de tipo CommandHandler.

Su handler es el siguiente:

deepsua_handlers.start_handler(update, context) ¶

Este comando envía un mensaje de bienvenida al usuario que lo lanza. Sólo funciona en el chat privado.

Comandos tipo: CallbackQueryHandler¶

Este tipo de handler tiene dos partes. La primera parte es el handler en sí, y la segunda parte es el callback que se ejecuta cuando se recibe una respuesta a un botón de un mensaje de tipo InlineKeyboardMarkup.
