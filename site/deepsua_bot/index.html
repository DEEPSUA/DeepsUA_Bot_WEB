deepsua_bot ¶

Este fichero es el encargado de gestionar el bot de Telegram.

Para ello, se encarga de crear el updater, el dispatcher y los handlers.

El updater es el encargado de recibir los mensajes del bot y el dispatcher es el encargado de enviarlos a los handlers.

Los handlers son los encargados de gestionar los mensajes recibidos por el bot. En este caso, se han creado dos handlers, uno para los comandos y otro para los mensajes de texto.

Para más información sobre los handlers, consultar la documentación de python-telegram-bot: https://python-telegram-bot.readthedocs.io/en/stable/

El código de los handlers se encuentra en el fichero deepsua_handlers.py

Para ejecutar el bot, se debe ejecutar el siguiente comando:

python3 deepsua_bot.py
